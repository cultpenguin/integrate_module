Dual data
================

Joint inversion is supported by using multiple data types.
The first example splits a single tTEM data set (with both low and high moment), into two separate data sets, one for low, and one for high.
These data can then be inverted seperately, or jointly.

.. nbgallery::
   notebooks/integrate_dual_data.ipynb
